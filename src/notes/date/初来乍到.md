# 开启杭漂之旅

一转眼就毕业了，甚至有点恍惚，周围的人陆陆续续备战考研、考公，而我还是决定了寻找一份合适的机遇（工作）最开始还是在校招投简历，刚开始没有回复，后续都是通知无薪培训一个月后上岗，我都拒了。。。

机缘只能另寻了，直接boss直骗（真多骗子），投了大概1000份简历，约上了6家杭州的面试，2家郑州面试（不想去其实），然后还有2家苏州的线上面，时机以至，起兵只身一人火车硬座直达美丽的大杭州
::: tip
杭州有面试邀约的话，是可以申请对应公司区域的人才公寓进行居住，可以省下一大笔酒店的钱，一定一定一定要提前3天预约，而且资料填写清楚不然会被打回，我在杭州玩了7天，没有提前申请，公寓只申请到了两天，亏麻了
::: 

到杭州的第一个周末，蹭了一顿学长的饭，头一次感受西湖的风景，感觉周围都很惬意（估计是当时人少，后来再去全是人头太挤了[点名银泰]）西湖醋鱼也没那么难吃，杭州的生活节奏也没有那么快，一切都是刚刚好

11-21日

面试了一家很特别的公司，没有八股，，充满活力，公司老总待人亲和，虽然实习生工资普遍低，但是却让我感觉到和其他公司的不同，拥抱人工智能，有不一样的发展轨迹，虽然吧，技术栈是Java和半吊子全栈，招收的是go后端开发以及全栈，咬咬牙还是可以做到的，意料之中的我被录用了，也认识了公司中另一个技术小哥。

希望未来的日子，我可以学到不一样的经验，体验新的生活，在这段实习的日子里，创造自己的价值，真正去编写维护产品，将各位大佬的智慧收入囊中【老总真的太有实力了，独到的想法和缜密的思维是我所仰慕的，真·大佬😭】
